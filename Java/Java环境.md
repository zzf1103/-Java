#Java环境配置
1.JDK是Java的开发工具包
  JRE是Java运行的环境
  JVM是把Java的字节码转成机器码并执行的虚拟机，使之可以在不同平台上运行。
  JDK包含JRE,JRE包含JVM
  由JVM翻译字节码到当前运行平台的机器码，再由JRE运行
2.JAVA-HOME和Path中的%JAVA-HOME%\bin
  path为命令行提供了执行Java的对应程序的路径范围，使命令行能在JDK的bin中找到
